# challenge-javascript-01 - Area of a Triangle

Escribe una función que tome la base y la altura de un triángulo y devuelva su área.

### Instalación
```
npm install
```

### test
```
npm run test
```

### RETO

Completa la función para calcular el área de un triángulo.
